ASKUE VZEP v4.0.0.49 (15.08.2014)
==============================
+ Добавлена возможность выбора сервера подключения к базе при запуске программы (локальный/удаленный)
+ Добавлена возможность выбора адресов для опроса по признаку "вид соединения" (CSD/GPRS)
* В зависимости от вида соединения адреса в главном окне программы различаются цветами (CSD - синий, GPRS - зеленый)
+ В настройках программы добавлена возможность включить или выключить дочитывание несчитанных счетчиков (по умолчанию дочитывание выключено)
